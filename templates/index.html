Cara membaca hasil ini
Deskripsi Akhir

Fungsi: output utama eksperimen sesuai mode yang dipilih.

Penentuan: Gemma Only memakai deskripsi citra. Sensor Assisted menerapkan koreksi per sensor lalu menambahkan rata-rata frontal ketika dua sensor memenuhi gate.

Batasan: ini deskripsi visual-spasial dari gambar 2D, bukan instruksi navigasi atau ukuran jarak presisi.

Verifikasi Sensor Frontal

Fungsi: menunjukkan pembacaan masing-masing HC-SR04, selisih pasangan, dan rata-rata yang dipakai.

Input & metode: nilai mentah Sensor 1 dan Sensor 2 dikoreksi secara terpisah, lalu dirata-ratakan ketika valid, segar, searah kamera, dan konsisten.

Batasan: cone sensor tidak memberi identitas objek dan tidak membuktikan jarak objek yang disebut Gemma.

Rata-rata Frontal

Fungsi: memberi satu referensi jarak pada arah dua sensor.

Input & metode: sistem menghitung (Sensor 1 terkoreksi + Sensor 2 terkoreksi) / 2 setelah seluruh gate pasangan lulus.

Batasan: partial, stale, konflik, atau beda arah membuat rata-rata ditahan.

Ringkasan Adegan

Fungsi: memisahkan informasi semantik gambar seperti objek utama, posisi, dan tipe adegan.

Penentuan: nilai berasal dari output terstruktur Gemma atau fallback deskripsi visual jika struktur tidak lengkap.

Batasan: model dapat keliru membaca objek kecil, objek tertutup, atau konteks adegan yang ambigu.

Detail Teknis

Fungsi: membaca biaya komputasi tiap tahap inference.

Penentuan: waktu Gemma, pemrosesan sensor, dan total dicatat saat alur analisis berjalan.

Batasan: latensi tidak menunjukkan kualitas deskripsi; angka dapat berubah karena perangkat, model, atau kondisi runtime.

JSON Mentah

Fungsi: menyediakan data lengkap untuk audit teknis dan pelacakan eksperimen.

Penentuan: isi berasal dari respons backend setelah analisis gambar selesai.

Batasan: ini informasi lanjutan untuk pelacakan teknis; pembaca utama sebaiknya memakai deskripsi akhir, metrik, dan insight.
